Sim, existem várias opções de softwares e bibliotecas gratuitas que permitem integrar gráficos de forma responsiva em dashboards. Abaixo estão algumas das melhores opções:

1. Chart.js
Descrição: Uma das bibliotecas JavaScript mais populares e gratuitas para gráficos. É leve, fácil de usar e altamente personalizável. Suporta diversos tipos de gráficos (barra, linha, pizza, radar, entre outros).
Responsividade: Totalmente responsivo e ajusta os gráficos automaticamente ao redimensionamento da tela.
Integração: Fácil de integrar com qualquer dashboard web, sendo apenas necessário incluir os arquivos JavaScript e definir o HTML do canvas para os gráficos.
Website: Chart.js

2. Google Charts
Descrição: Ferramenta gratuita oferecida pelo Google, com suporte para uma ampla variedade de gráficos e visualizações de dados. É baseado em JavaScript e fácil de implementar em qualquer aplicação web.
Responsividade: Oferece gráficos responsivos com opções de personalização para ajustar bem em dispositivos móveis.
Integração: Funciona bem em dashboards com integração via JavaScript e HTML.
Website: Google Charts

3. ApexCharts
Descrição: ApexCharts é uma biblioteca de gráficos moderna, focada em gráficos responsivos e interativos. É altamente personalizável, com suporte para gráficos de linha, área, barra, pizza e muito mais.
Responsividade: Projetada para ser responsiva e pode ser usada em dashboards com uma interface adaptável.
Integração: Facilmente integrável em frameworks modernos como React, Vue, Angular, ou em aplicações simples de JavaScript.
Website: ApexCharts

4. Plotly
Descrição: Plotly é uma biblioteca poderosa que suporta gráficos interativos e visualizações complexas. É muito popular para dashboards e visualizações de dados avançadas, com gráficos responsivos.
Responsividade: Gráficos interativos e responsivos por padrão.
Integração: Pode ser integrada com diversas plataformas e linguagens, como Python, JavaScript, e frameworks modernos (React, Vue).
Website: Plotly

5. D3.js
Descrição: D3.js é uma biblioteca JavaScript poderosa para criar gráficos dinâmicos e altamente customizáveis. Embora tenha uma curva de aprendizado maior, oferece flexibilidade total para criar gráficos únicos.
Responsividade: Totalmente responsivo, com gráficos que podem ser ajustados conforme o tamanho da tela.
Integração: Usada principalmente em aplicações JavaScript e é uma excelente opção para dashboards personalizados.
Website: D3.js

6. Datawrapper
Descrição: Datawrapper é uma ferramenta online gratuita para criar gráficos interativos, mapas e tabelas. Embora não seja uma biblioteca de código, ele oferece uma interface simples para criar gráficos que podem ser integrados diretamente em dashboards.
Responsividade: Todos os gráficos gerados são responsivos e prontos para dispositivos móveis.
Integração: Os gráficos podem ser incorporados em qualquer dashboard através de código embutido (embed code).
Website: Datawrapper

Melhor escolha:
Para simplicidade e facilidade de uso, o Chart.js ou Google Charts são ótimas opções. Eles são fáceis de integrar e personalizar.
Se você precisa de gráficos mais interativos e avançados, o ApexCharts ou Plotly podem ser mais adequados.
Essas ferramentas oferecem bastante flexibilidade para criar dashboards interativos e responsivos, e todas são gratuitas para uso básico.
